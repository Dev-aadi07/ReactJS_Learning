# Responsive Navbar Layout

A responsive navbar built using React + Tailwind CSS as part of a practical React learning journey.

## Concepts Covered

* Flexbox Layouts
* Nested Flex Containers
* Dynamic Rendering using `map()`
* React `key` Prop
* Tailwind Spacing Utilities
* Hover Effects
* Responsive Wrapping
* `useState` Hook
* Conditional Styling
* Event Handling

---

## Features

* Responsive navbar layout
* Dynamic navigation items
* Active link highlighting
* Smooth hover transitions
* Clean flexbox structure
* Reusable Tailwind classes

---

## Tech Stack

* React
* Vite
* Tailwind CSS

---

## Learning Focus

This mini project focuses on understanding:

* layout structuring using flexbox
* how React dynamically renders UI
* managing UI state using `useState`
* conditional class rendering
* building reusable and scalable navbar structures
