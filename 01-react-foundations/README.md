# React Foundations

A beginner-to-advanced React learning journey focused on building muscle memory through small practical projects instead of tutorial-based learning.

## Concepts Covered

* Functional Components
* JSX
* Props
* Reusable Components
* Dynamic Rendering using `map()`
* `key` Prop
* Tailwind CSS Basics
* Flexbox Layouts
* Visual Hierarchy
* Card UI Design
* Hover Effects

---

## Mini Project

### Developer Team Section

A responsive developer team UI built using React + Tailwind CSS.

### Features

* Reusable `ProfileCard` component
* Dynamic rendering using array data
* Responsive flex layout
* Uniform card sizing
* Typography hierarchy
* Hover animations
* Tailwind utility styling
